Manufacturing Plant FP&A Dashboard

Power BI · operational FP&A · variance bridge

Operational FP&A dashboard for a fictional diversified industrial manufacturer (4 plants, 24 months). Plant-level P&L, a price/volume/mix waterfall, OEE and scrap, and inventory turns. The data has a steel-cost shock and one underperforming plant built in so there's something to flag.

Industrial Margin Bridge

Python · pandas · SEC EDGAR

Five-year margin walk on real 10-K segment data from a diversified industrial. Decomposes YoY gross-margin changes into price, volume, mix, and cost effects. Built in Python with pandas, output as a waterfall with written commentary.

Manufacturing Operations SQL

SQL · CTEs · window functions

A synthetic ERP-style schema (production, scrap, BOMs, inventory) with six business questions answered using CTEs and window functions. Things like scrap-rate outliers by plant, on-time delivery by SKU family, and working-capital efficiency.

Demand Forecast & Production Plan

R · ARIMA · Holt-Winters · optimization

Quarterly unit-demand forecast for an industrial product line using ARIMA and Holt-Winters, then translated into a production schedule that respects plant capacity. Written up in RMarkdown with diagnostics and forecast intervals.
